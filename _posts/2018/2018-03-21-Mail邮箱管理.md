## 添加账号

#### QQ 邮箱

登录 QQ 邮箱网页版，点击设置->账号->POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV 服务

开启 IMAP/SMTP 服务

| 协议 | 服务器地址  |
| ---- | ----------- |
| POP3 | pop.qq.com  |
| IMAP | imap.qq.com |
| SMTP | smtp.qq.com |
